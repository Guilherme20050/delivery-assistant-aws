# Delivery Assistant com AWS Step Functions e Bedrock

## Descrição
Projeto desenvolvido durante o bootcamp da DIO com o objetivo de criar um assistente inteligente para gerenciamento de pedidos de delivery utilizando AWS.

## Arquitetura
- AWS Step Functions (orquestração)
- AWS Lambda (lógica de negócio)
- Amazon Bedrock (IA)

## Fluxo do Sistema
1. Recebimento do pedido
2. Validação do pedido
3. Processamento do pagamento
4. Atualização do status
5. Geração de resposta com IA

## Tecnologias
- Python
- AWS Lambda
- AWS Step Functions
- Amazon Bedrock

## Resultados
Sistema capaz de automatizar o fluxo de pedidos com resposta inteligente ao cliente.
